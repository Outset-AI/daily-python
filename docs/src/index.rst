Daily Client SDK for Python
=======================================

Welcome to `Daily`_'s Python client SDK API reference!

The Daily Client SDK for Python allows you to build video and audio calling into
your native desktop and server applications.

You can find installation, reference guides and a bunch of cool demos in our
`reference docs <https://docs.daily.co/reference/daily-python>`_.
